# HW2 - Real Estate Listing Description Generator

## Workflow Chosen
Generate professional real estate listing descriptions from structured property notes.

## User
Real estate agents, leasing agents, or homeowners preparing property listings.

## Input
Structured or semi-structured property details, such as:
- Number of bedrooms/bathrooms
- Square footage
- Features (e.g., updated kitchen, finished basement)
- Amenities
- Location highlights

## Output
A polished, market-ready listing description suitable for platforms like Zillow, MLS, or rental sites.

## Why This Task is Valuable
Writing listing descriptions is time-consuming and inconsistent. Automating the first draft improves efficiency, ensures a professional tone, and allows users to quickly generate high-quality listings while still enabling human review for accuracy and compliance.
